# Emirati Pocket v2 🌙
### English · Manglish · Arabic — Emirati Dialect Translator

---

## Files to upload to GitHub
- `index.html` — main app
- `manifest.json` — PWA config
- `sw.js` — service worker (offline support)

---

## Setup on GitHub Pages

1. Create new repo on github.com → name it `emirati-pocket` → **Public**
2. Upload all 3 files above
3. Go to **Settings → Pages → Source → main → Save**
4. Wait 1-2 min → your app is live at:
   `https://YOUR-USERNAME.github.io/emirati-pocket`

## Install on Phone

**iPhone:** Safari → Share → Add to Home Screen  
**Android:** Chrome → 3 dots → Add to Home Screen

---

## Get Free Gemini API Key

1. Go to **aistudio.google.com**
2. Sign in with Google
3. Click **Get API Key** → **Create API Key**
4. Copy it (starts with `AIza...`)
5. Open the app → paste in the key box → Save

## Features

| Feature | Details |
|---------|---------|
| 🇬🇧 English → Emirati | Full grammar notes, examples |
| 🇮🇳 Manglish → Emirati | Type Malayalam in English letters |
| 🔄 Arabic → Malayalam | Paste any Arabic, get Malayalam |
| 🎤 Voice Input | Speak in English or Malayalam |
| 🔊 Voice Output | Hear the translation spoken aloud |
| 📜 History | Last 60 translations saved on device |
| 📴 PWA | Installs on home screen, works offline (UI) |
